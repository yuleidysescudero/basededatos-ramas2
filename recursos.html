SQL Básico

-- Consulta de ejemplo
SELECT c.nombre, SUM(f.total) AS ventas
FROM clientes c
JOIN facturas f ON f.id_cliente = c.id
GROUP BY c.nombre
ORDER BY ventas DESC;

Buenas Prácticas

  • Tipos correctos y restricciones (PK, FK, UNIQUE, CHECK).
  • Indexa campos consultados frecuentemente.
  • Evita SELECT * en producción.
  • Respaldos y pruebas de recuperación.
